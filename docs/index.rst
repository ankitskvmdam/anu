Predicting interactions among protein, DNA and RNA.

Installation
------------

To install Anu,
run this command in your terminal:

.. code-block:: console

   $ pip install anu


Usage
-----

Anu's usage looks like:

.. code-block:: console

   $ anu [OPTIONS]

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
